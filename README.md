# 🌸 Spring XML Configuration Demo

## 📌 Overview

A Spring Framework project demonstrating **XML-based dependency injection** using a Payment Service system.

This project showcases how Spring manages object creation and dependency injection using an `ApplicationContext.xml` configuration file.

---

## 🚀 Features

- ✅ XML-based Spring configuration
- ✅ Multiple Payment Methods:
  - UPI
  - Debit Card
  - Credit Card
- ✅ Expense Tracker with Dependency Injection
- ✅ Strategy Design Pattern implementation
- ✅ Clean separation of concerns

---

## 🧠 Technologies Used

- Java 21
- Spring Framework 7.0.3
- Maven
- Object-Oriented Programming (OOP)
